The Snake

The snake consists of a list with each segment defined by a dictionary with the x and y

We can draw the snake using the filled_rect

 

import pgzrun

 

# Variables

TITLE = "Snake"

WIDTH = 300

HEIGHT = 250

 

cell_size = 15

 

snake = [

        {"x": 5, "y": 0},

        {"x": 4, "y": 0},

        {"x": 3, "y": 0},

        ]

 

# pgzero functions

def draw():

        screen.clear()

 

        for segment in snake:

            cell = Rect(

                     segment["x"] * cell_size,

                     segment["y"] * cell_size,

                     cell_size - 1,

                     cell_size - 1)

            

             screen.draw.filled_rect(

                cell,

                "LIGHT GREEN"                

                );

 

pgzrun.go()

Moving the Snake

 

We will use the pgzero update function to move the snake segments.
First we will move the snake right, which we can do by increasing each one of the segments x component by one.

We will insert the new segment to the start of the snake list. The last element of the list is removed with the `pop` list method.

 

def update():

        

        new_x_pos = snake[0]["x"] + 1

        new_y_pos = snake[0]["y"]

        

        snake.insert(0, {"x": new_x_pos, "y": new_y_pos})

        snake.pop()

 

When we run the above code, we see the snake move to the right of the screen and then disappear.

However, it is moving too fast. This is because the update function is called 60 times a second. We can control this by passing a single parameter to the update function. We will use this parameter to scale the movement speed.

 

FPS = 15

 

# [...]

 

timer = 0

 

# pgzero functions

def update(dt):

        global timer

        

        new_x_pos = snake[0]["x"] + 1

        new_y_pos = snake[0]["y"]

        

        timer += dt

        if timer >= FPS / 60:

        timer = 0

        snake.insert(0, {"x": new_x_pos, "y": new_y_pos})

        snake.pop()

 

 

 

By introducing the timer, we can control how often the snake segments are updated.
